JavaScript Nesne yönetimleri

var person = {

firstName: "John",

lastName : "Doe",

id : 5566,

fullName : function() {

return this.firstName + " " + this.lastName;

}

};


This kelime

İşlev tanımında, işlevin this"sahibine" karşılık gelir.

Yukarıdaki örnekte, thisbir kişi nesnesi "sahibi olan" fullName fonksiyonu.

Başka bir deyişle, this.firstName bu nesnenin firstName özelliği anlamına gelir .

JS'de bu Anahtar Kelime'dekithis anahtar kelime hakkında daha fazla bilgi edinin .

JavaScript Yöntemleri

JavaScript yöntemleri, nesneler üzerinde gerçekleştirilebilecek eylemlerdir.

JavaScript yöntemi , işlev tanımı içeren bir özelliktir .

EMLAK DEĞER
İsim John
Soyadı aytekin
yaş 50
göz rengi mavi
ad soyad function () {return this.firstName + "" + this.lastName;}
Yöntemler, nesne özellikleri olarak depolanan işlevlerdir.

Nesne Yöntemlerine Erişim

Bir nesne yöntemine aşağıdaki sözdizimiyle erişirsiniz:

objectName.methodName()

Genellikle fullName () öğesini person nesnesinin bir yöntemi olarak ve fullName özelliğini de bir özellik olarak tanımlarsınız.

FullName özelliği () ile çağrıldığında (işlev olarak) yürütülür.

Bu örnek , bir kişi nesnesinin fullName () yöntemine erişir :

ÖRNEK:name = person.fullName();

FullName özelliğine () olmadan erişirseniz , işlev tanımını döndürür :

ÖRNEK:name = person.fullName;

FullName özelliğine () olmadan erişirseniz , işlev tanımını döndürür :

ÖRNEK:name = person.fullName;

Yerleşik Yöntemleri Kullanma

Bu örnek, toUpperCase()bir metni büyük harfe dönüştürmek için String nesnesinin yöntemini kullanır :

var message = "Hello world!";

var x = message.toUpperCase();

Yukarıdaki kod yürütüldükten sonra x değeri:

HELLO WORLD!

Bir Nesneye Yöntem Ekleme

Bir nesneye yeni bir yöntem eklemek kolaydır:

person.name = function () {

return this.firstName + " " + this.lastName;

};
