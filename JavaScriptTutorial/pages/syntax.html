JavaScript Sözdizimi

JavaScript sözdizimi, JavaScript programlarının nasıl oluşturulduğu ile ilgili kurallar dizisidir:

ÖRNEK:

  • var x, y, z; // How to declare variables

  • x = 5; y = 6; // How to assign values

  • z = x + y; // How to compute values

    • JavaScript Değerleri

    JavaScript sözdizimi iki tür değer tanımlar: Sabit değerler ve değişken değerler.

    Sabit değerlere değişmez değerler denir . Değişken değerlerine değişken denir.

    JavaScript Değişmezleri

    Sabit değerler yazmak için en önemli kurallar şunlardır:

    Sayılar ondalıklı veya ondalıksız olarak yazılır:

    ÖRNEK:
  • 10.50

  • 1001

    • Dizeler çift ​​veya tek tırnak içinde yazılan metindir:

    ÖRNEK:
  • "John Doe"

  • 'John Doe'

    • JavaScript Değişkenleri

    Bir programlama dilinde, veri değerlerini saklamak için değişkenler kullanılır .

    JavaScript, değişkenleri bildirmek için varanahtar kelimeyi kullanır .

    Bir eşittir işareti kullanılır atama değerlere değişkenlere.

    Bu örnekte, x bir değişken olarak tanımlanmıştır. Sonra, x değeri 6'ya atanır (verilir):

    ÖRNEK:
  • var x;

  • x = 6;

    • JavaScript Operatörleri

    JavaScript, değişkenlere değer atamak için bir atama operatörü ( =) kullanır :

    JavaScript değerleri hesaplamak için aritmetik işleçleri ( + - * /) kullanır :

    ÖRNEK: (5 + 6) * 10

    JavaScript Yorumları

    Tüm JavaScript ifadeleri "yürütülmez".

    Çift bölü sonra Kod //veya aralarında /*ve */bir olarak kabul edilir comment .

    Yorumlar yok sayılır ve yürütülmez.

    JavaScript İfadeleri

    İfade, bir değeri hesaplayan değerlerin, değişkenlerin ve işleçlerin birleşimidir.

    Hesaplamaya değerlendirme denir: 5*10

    JavaScript Anahtar Kelimeleri

    JavaScript anahtar sözcükleri , gerçekleştirilecek eylemleri tanımlamak için kullanılır. varAnahtar kelime değişkenler oluşturmak için tarayıcı söyler:

    ÖRNEK:var x, y;

    x = 5 + 6;

    y = x * 10;
