JavaScript HTML DOM Öğeleri

Bu sayfa size bir HTML sayfasındaki HTML öğelerini nasıl bulacağınızı ve erişeceğinizi öğretir.

HTML Öğelerini Bulma

Genellikle JavaScript ile HTML öğelerini değiştirmek istersiniz.

Bunu yapmak için önce elemanları bulmalısınız. Bunu yapmanın birkaç yolu vardır:

  • HTML öğelerini kimliğe göre bulma
  • Etiket adına göre HTML öğeleri bulma
  • Sınıf adına göre HTML öğeleri bulma
  • CSS seçicileri ile HTML öğeleri bulma
  • HTML nesnesi koleksiyonlarına göre HTML öğelerini bulma
    •

    Kimliğe Göre HTML Öğesi Bulma

    DOM'de bir HTML öğesi bulmanın en kolay yolu, öğe kimliğini kullanmaktır.

    Bu örnek şu öğeyi bulur id="intro":

    ÖRNEK:var myElement = document.getElementById("intro");

    Öğe bulunursa, yöntem öğeyi nesne olarak döndürür (myElement öğesinde).

    Öğe bulunamazsa, myElement içerecektir null.

    Etiket Adına Göre HTML Öğelerini Bulma

    Bu örnek tüm

    öğeleri bulur :

    ÖRNEK:var x = document.getElementsByTagName("p");

    Bu örnekte öğeyi bulur ve içindeki id="main"tüm

    öğeleri bulur "main":

    ÖRNEK:var x = document.getElementById("main");
    var y = x.getElementsByTagName("p");

    Sınıf Adına Göre HTML Öğelerini Bulma

    Aynı sınıf adına sahip tüm HTML öğelerini bulmak istiyorsanız, kullanın getElementsByClassName().

    Bu örnek, ile tüm öğelerin bir listesini döndürür class="intro".

    ÖRNEK: var x = document.getElementsByClassName("intro");
    Sınıf adına göre öğe bulma, Internet Explorer 8 ve önceki sürümlerinde çalışmaz.

    CSS Seçicileri ile HTML Öğeleri Bulma

    Belirli bir CSS seçicisiyle eşleşen tüm HTML öğelerini (id, sınıf adları, türleri, öznitelikleri, öznitelik değerleri, vb.) Bulmak istiyorsanız, querySelectorAll()yöntemi kullanın .

    ÖRNEK:var x = document.querySelectorAll("p.intro");
    querySelectorAll()Yöntem Internet Explorer 8 ve önceki sürümlerinde çalışmaz.
