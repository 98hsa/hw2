JavaScript HTML DOM - CSS'yi değiştirme

HTML DOM, JavaScript'in HTML öğelerinin stilini değiştirmesine izin verir.

HTML Stilini Değiştirme

Bir HTML öğesinin stilini değiştirmek için şu sözdizimini kullanın:

document.getElementById(id).style.property = new style

Aşağıdaki örnek bir <.p> öğenin stilini değiştirir :

<.html>

<.body>

<.p id="p2">Hello World!<./p>

<.script>

document.getElementById("p2").style.color = "blue";

<./script>

<.p>The paragraph above was changed by a script.<./p>

<./body>

<./html>

Olayları Kullanma

HTML DOM, bir olay meydana geldiğinde kod yürütmenize olanak tanır.

Etkinlikler, HTML öğelerinde "bir şeyler olduğunda" tarayıcı tarafından oluşturulur:

  • Bir öğe tıklandı
  • Sayfa yüklendi
  • Giriş alanları değiştirildi

    • Bu öğreticinin bir sonraki bölümünde olaylar hakkında daha fazla bilgi edineceksiniz.

    Bu örnek id="id1", kullanıcı bir düğmeyi tıklattığında HTML öğesinin stilini değiştirir :

    <.!DOCTYPE html>

    <.html>

    <.body>

    <.h1 id="id1">My Heading 1<./h1>

    <.button type="button"<./p>

    onclick="document.getElementById('id1').style.color = 'red'">

    Click Me!<./button>

    <./body>

    <./html>
